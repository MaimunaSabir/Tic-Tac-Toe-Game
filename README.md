# Tic-Tac-Toe-Game
A simple and fun Tic Tac Toe game built using HTML, CSS, and JavaScript.
This project was created for practice to improve my front-end development skills. It’s a two-player game where players take turns to mark X and O on a 3×3 grid.
    
    Features
Two-player gameplay
Win and draw detection
Restart game button
Clean and simple UI
